# Герой на букву — Dota 2 рулетка

Красивая веб-аппка для пати: крутишь — получаешь случайного героя Dota 2 + случайную букву русского алфавита. Нужно описать героя, начиная с этой буквы.

## Особенности

- 🔥 Красивая анимация прокрутки (два слот-машинных барабана)
- 🎵 Звуки тиканья и динга (Web Audio API)
- ✨ Конфетти и плавные easing'и
- 📜 История спинов + сохранение своих описаний (localStorage)
- 📱 Отлично выглядит на телефоне и десктопе
- 🐍 Полностью на Python (FastAPI) + современный фронтенд без сборки
- 127 героев + 33 буквы

## Запуск локально

```bash
cd dota-bukva

# Создай venv (рекомендуется)
python -m venv venv
source venv/bin/activate   # Linux/Mac
# или venv\Scripts\activate  # Windows

pip install -r requirements.txt

python app.py
# или
uvicorn app:app --reload
```

Открой http://localhost:8000

## Как это работает

- `GET /` — отдаёт красивую HTML-страницу (Jinja)
- `GET /api/spin` — Python выбирает случайного героя + букву и отдаёт JSON
- Фронтенд (ванильный JS + Tailwind) получает результат и **анимационно прокручивает** барабаны так, чтобы выпасть именно на нужный герой и букву

Анимация написана через Web Animations API + requestAnimationFrame-подобный подход, очень плавная (60fps).

## Для деплоя на Vercel

Vercel поддерживает Python через serverless functions.

### Самый простой способ (рекомендую):

1. Установи Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. В папке `dota-bukva` выполни:
   ```bash
   vercel
   ```

3. При первом деплое создаст `vercel.json` примерно такого вида:

```json
{
  "builds": [
    { "src": "app.py", "use": "@vercel/python" }
  ],
  "routes": [
    { "src": "/(.*)", "dest": "app.py" }
  ]
}
```

Или просто используй чистый статический деплой + одну serverless-функцию:

- Перенеси фронтенд в `public/index.html`
- Сделай `api/spin.py` (Vercel Python function)
- Это даст идеальную производительность и бесплатный хостинг

Если хочешь — могу быстро сделать отдельную версию под чистый Vercel static + serverless.

## Структура проекта (готово к Vercel)

```
dota-bukva/
├── app.py                 # FastAPI локально (грузит данные из data/)
├── api/
│   └── spin.py            # Vercel Serverless функция (тот же рандом)
├── data/
│   └── heroes.json        # Единственный источник данных героев (126+)
├── public/
│   └── index.html         # Статическая сборка фронтенда для Vercel
├── templates/
│   └── index.html         # Исходник фронтенда (для локальной разработки)
├── vercel.json            # Конфиг роутинга и сборки для Vercel
├── requirements.txt
└── README.md
```

## Деплой на Vercel (одной командой)

1. Установи Vercel CLI (если ещё нет):
   ```bash
   npm i -g vercel
   ```

2. В папке проекта:
   ```bash
   vercel
   ```

   При первом запуске залогинься и ответь на вопросы (рекомендуется "Yes" на все для дефолта).

3. Готово! Твой сайт будет на `https://твоё-имя.vercel.app`

   При каждом `git push` или `vercel --prod` будет новый деплой.

Frontend отдаётся статически (молниеносно), `/api/spin` — лёгкая Python-функция на сервере Vercel.

Если хочешь полностью без сервера (чистый клиентский рандом) — скажи, сделаю версию в один HTML-файл.
